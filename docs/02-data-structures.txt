============================================
  BrotatoLike 数据结构协议
============================================

一、ECS 组件（纯数据，无逻辑）

TransformComponent:
  - position: Vector2         # 世界坐标位置

HealthComponent:
  - current_hp: int
  - max_hp: int
  - invincible_time: float     # 无敌帧时间
  - invincible_timer: float

MovementComponent:
  - speed: float              # 移动速度（像素/秒）
  - direction: Vector2        # 移动方向（单位向量）
  - is_player: bool            # 是否玩家

WeaponComponent:
  - weapon_id: String
  - cooldown_remaining: float
  - base_cooldown: float
  - damage: int
  - range: float
  - bullet_speed: float
  - bullet_size: float
  - bullet_count: int
  - spread: float
  - targeting: String          # "nearest" / "player"
  - bullet_shape: String
  - bullet_color: String
  - melee: bool                # 是否近战武器
  - melee_type: String         # "slash"/"thrust"/"blunt"

CollisionComponent:
  - radius: float

SpriteComponent:
  - shape: String              # circle/rect/diamond/triangle/hexagon/barrel/arc/composite
  - color: Color
  - size: float
  - rotation: float
  - outline_radius: float      # 黑色描边大小
  - height: float              # 伪3D高度系数（>0.8绘制地面阴影）
  - sub_sprites: Array         # 复合子精灵

LifetimeComponent:
  - remaining_time: float

PlayerTag: { }
EnemyTag: {
  - enemy_id: String
  - damage: int
  - xp_drop: int
  - coin_drop: int
  - is_boss: bool
  - base_speed: float
}
ProjectileTag:
  - owner_id: int
  - damage: int
  - pierce: int                # 穿透次数
XPOrb:
  - xp_value: int              # 经验值

二、配置数据结构

EnemyConfig:
  - id, name, shape, color, hp, speed, damage, size
  - weapon_id, xp_drop, coin_drop, is_boss
  - on_death: String (可空)

WeaponConfig:
  - id, name, ecosystem, tier (1/2/3)
  - cooldown, damage, range, bullet_speed, bullet_size
  - bullet_count, spread, bullet_shape, bullet_color, targeting
  - melee: bool, melee_type: String, price, element, explosion

ItemConfig:
  - id, name, desc, type, stat_key, stat_value, price
  - stat_key2, stat_value2 (可选第二属性)

CharacterConfig:
  - id, name, desc
  - starting_weapons: Array
  - stat_modifiers: Dict     # 角色初始属性加成(含负值)

PerkConfig:
  - id, name, desc
  - modifiers: Array[Dict]   # 多属性修改
  - rarity: int (1默认/2稀有)

WaveConfig:
  - wave_number, enemy_groups ("id:weight,...")
  - spawn_interval, duration, is_boss_wave, boss_id

三、玩家属性（20项）
  max_hp, hp_regen, speed, damage, damage_percent
  attack_speed, crit_chance, crit_damage, armor, dodge_chance
  range, luck, xp_gain, coin_gain, pickup_radius
  knockback, pierce, life_steal, explosion_chance, explosion_size

四、运行时数据（GameManager）
  - state: 7状态 (MENU/CHAR_SELECT/PLAYING/PERK_SELECT/SHOP/WIN/LOSE)
  - coins, current_wave, player_level, player_exp, exp_to_next
  - kills, owned_items, owned_weapons, selected_character
  - pending_levels, perk_choices, stat_bonuses (20项)

五、联机数据（MultiplayerManager）
  - is_host: bool, is_online: bool
  - peer: ENetMultiplayerPeer
  - connected_players: Dictionary
  - DEFAULT_PORT: 9876, MAX_PLAYERS: 4
