============================================
  BrotatoLike 游戏流程图
============================================

┌──────────────┐
│   游戏启动    │
└──────┬───────┘
       ▼
┌──────────────┐
│  主菜单界面   │
│ [单人模式]    │ → GameManager.reset() → CHAR_SELECT
│ [联机模式]    │ → GameManager.reset() → LOBBY
│ [退出游戏]    │
└──────────────┘

联机模式 ──→ LOBBY (lobby.tscn)
               ├── [创建主机] → ENetServer port 9876
               ├── [加入游戏] → 输入IP → ENetClient连接
               └── 连接成功后 → CHAR_SELECT

┌──────────────┐
│  角色选择     │  ← character_select.tscn
│  13角色卡片   │
│  点击选择角色 │  → GameManager.apply_character(id)
└──────┬───────┘  → start_game() → PLAYING
       ▼
┌──────────────┐
│  战斗场景     │  ← game_world.tscn
│  波次 N=1    │
└──────┬───────┘
       ▼
┌──────────────┐
│  波次进行中   │
│  持续出怪     │  SpawnSystem: 全图随机（离玩家350px+）
│  玩家走位     │  MovementSystem + 敌人互相避让
│  自动攻击     │  WeaponSystem: 暴击/吸血/穿透/近战动画
│  击杀敌人     │  HealthSystem: 掉落金币+经验球(diamond)
│  拾取经验球   │  game_world._collect_xp_orbs(pickup_radius)
│  倒计时结束   │
└──────┬───────┘
       │
       ├─── 每 5 波 Boss ───→ 随机Boss (11种循环)
       │
       ▼
┌──────────────┐
│  波次结束     │
└──────┬───────┘
       │
       ├─── 有 pending_levels ──→ PerkSelect(3选1) → perks_done
       │
       ├─── 联机: 复活死亡玩家(半血) ──→
       │
       ├─── N == 20  ──→ ResultsOverlay (胜利)
       │
       └─── 商店 shop_overlay
                ├── 左: 道具+武器购买+刷新
                ├── 右: 背包(ScrollContainer)+▲▼交换+合成+出售
                └── 右下: 金币+下一波

游戏状态机:
  MENU → CHAR_SELECT/LOBBY → PLAYING → SHOP → PLAYING → ...
  PLAYING → PERK_SELECT → SHOP → PLAYING → ...
  → WIN/LOSE → MENU

联机规则:
  - Host 运行完整 ECS 模拟
  - 客户端发送输入，Host 回复状态
  - 死亡不立即结束，全员死亡才GameOver
  - 波次结束自动半血复活死亡玩家(2秒无敌)
