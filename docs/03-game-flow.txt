============================================
  BrotatoLike 游戏流程图
============================================

┌──────────────┐
│   游戏启动    │
└──────┬───────┘
       ▼
┌──────────────┐
│  主菜单界面   │  ← MainMenu.tscn
│  [开始游戏]   │
└──────┬───────┘
       │ 点击开始
       ▼
┌──────────────┐
│  战斗场景     │  ← GameWorld.tscn
│  波次 N=1    │
└──────┬───────┘
       │ 初始化波次
       ▼
┌──────────────┐
│  波次进行中   │
│  生成敌人 →   │  SpawnSystem 按波次配置生成
│  玩家走位 →   │  MovementSystem 处理输入
│  自动攻击 →   │  WeaponSystem 自动瞄准+射击
│  击杀敌人 →   │  HealthSystem 处理死亡，掉落金币/经验
│  倒计时结束   │
└──────┬───────┘
       │
       ├─── 每 5 波 ───→ Boss 波次
       │
       ▼
┌──────────────┐
│  波次结束     │  判断结果
└──────┬───────┘
       │
       ├─── 玩家死亡 ──→ 游戏结束 → 返回主菜单
       │
       ├─── N == 30  ──→ 胜利 → 返回主菜单
       │
       └─── 其他 ──→ 商店界面 ShopOverlay.tscn
                        │
                        ├── 购买道具（4个随机道具）
                        ├── 购买武器（3个随机武器）
                        ├── 回复生命
                        └── 下一波
                              │
                              ▼
                        N += 1，回到战斗


四、游戏状态机

  MENU → PLAYING → SHOP → PLAYING → ... → WIN/LOSE → MENU

  状态说明:
    MENU:      主菜单
    PLAYING:   战斗中（波次进行中）
    SHOP:      商店界面
    WIN:       通关
    LOSE:      失败


五、波次规则

  - 共 30 波
  - 每 5 波随机出现一个 Boss（第 5, 10, 15, 20, 25, 30 波）
  - 普通波次持续 30 秒
  - Boss 波次持续 60 秒（或击杀 Boss 后结束）
  - 波次难度逐渐递增（敌人数量和属性增长）
